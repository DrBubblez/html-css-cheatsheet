HTML & CSS Cheat Sheet

Table Tags

Tag Name Description
<table> Table The wrapper element for all the content of the table.
<thead> Table Head The wrapper element for table rows that contain the column labels of the table.
<tbody> Table Body The wrapper element for table rows that contain the main content of the table.
<tfoot> Table Footer A wrapper element that typically contains summary or footer rows for the table.
<tr> Table Row The wrapper element for table cells in a row. It is a child of thead, tbody, or tfoot.
<th> Table Header Table header cell that labels the content of the table row or column.
<td> Table Data Table data cell that contains the main content of the table.

Table Attributes

Tag Name Description
colspan Column Span Defines the amount of columns the table data should span.
rowspan Row Span Defines the amount of rows the table data should span.
